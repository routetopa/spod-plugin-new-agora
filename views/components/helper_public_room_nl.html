Public room helper
Public room is a place when the users can discuss about previously created topic in the Agora area.
1
About the room
To the centre, rooms display a threaded chat (enabled for the use of open data). Moving to the far right, there is an area dedicated to Graphs, Search and Weekly messages notifications.
Moving to the far left, a vertical toolbar allows the following functionalities:
  • Weekly Messages Notifications
    • An user can check the number of unread messages and on the right top side, only unread messages will be displayed for every day of the week.
  • Search
    • It allows to find all of the massages posted by a particular person.
  • Sentiments
    • It shows a representation to display two values that represent the amount, represented by the percentage, of negative sentiment and positive comments.
  • Users Graph
    • It represents the relationship among users.
  • Datalets Graph
    • It represents the datalet on the left side and the datasets on the right side, edges represent relationships between them.
  • Comments Graph
    • It highlights the interactions among users in a discussion room.
  • E-mail Notification
    • It allows to keep track of the activity in each discussion with notifications via e-mail.
    2
    Threaded chat

    The threaded chat allows to reply to a discussion clicking on the "reply" link. Every time you add a new post in the chat, you can express you opinion by selecting from the Opinion Button :

    • Neutral ( Gray is color avatar highlighted in the discussion room).
    • Agree ( Green is color avatar highlighted in the discussion room).
    • Desagree ( Red is color node highlighted in the discussion room).
    3
    Interactive Graphs
    Any graph is interactive: you can navigate the nodes as you like, by clicking the nodes. Any graph is informative: with mouse over the node, related information is shown on top right. Any graph is synchronized: by clicking on a node in the graph you can see on the left the corresponding content in the threaded chat. A few graphs are:
    • Users Graph: graph of users, where each node represents a user that is participating in the discussion, and an edge joins to users that they have a friendship relations on the SPOD platform. If you bring the mouse over a node (no click) you can see the friends relationship among the users.
    • Datalets Graph: The graph represents two sets, on the left side the datalet and on the right side the datasets, edges represent relationships between them. If you bring the mouse over a node (no click) you can read the comment of the node and if you click the chat on the left will move and highlight the corresponding datalet with comment.
