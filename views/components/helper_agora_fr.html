Tuto de l'Agora
L' Agora est l'environnement qui permet à chaque modérateur de créer une «salle publique» pour commencer une nouvelle discussion avec/entre tous les utilisateurs. Une fois qu'une salle est créée, seul l'administrateur peut la supprimer. En haut à gauche de la page, l' Agora présente différents onglets pour visualiser les salles de discussion selon différents critéres (les plus récentes, les plus pertinentes, les plus populaires et mes salles), enfin une fonction de «recherche» est disponible. Sur le côté droit, la page affiche d'autres détails pour chaque salle de discussion sélectionnée (par exemple, description, notifications, statistiques, etc.). (Plus de détails ci-dessous).
1
Création d'une salle
Pour créer une nouvelle salle , appuyez simplement sur le grand bouton "+" (Ajouter) en bas, puis insérez le sujet et une brève description de la salle.
2
A propos des salles (1)
Sur le côté gauche, l' Agora affiche différentes options sur chaque onglet pour organiser les salles de discussion
  • "Dernièr" : Derniére salle consultée
  • "Pertinant" : La salle plus pertinente
  • "Populaire" : La salle plus populaire
  • "Mes Agora" : Sa(ses) propre(s) salle(s) de discussion créée(s)
  • Rechercher le titre ou la description d'une salle de discussion
    • 3
    A propos des salles (2)

    Sur le côté droit, plus de détails sont affichés pour chaque salle de discussion sélectionnée:

    • Description montre une courte description de la salle de discussion
    • La section Statistiques affiche Vues , Commentaires et Datalets . Affichages" montre Popularité: nombre de visites et commentaires
    • Commentaires et Datalets décrit respectivement le nombre de commentaires et de données dans une salle de discussion.
    • Utilisateurs indique le nombre d'utilisateurs qui ont participé à la discussion.
    • Les jeux de données suggérés montrent tous les jeux de données recommandés par les utilisateurs.
    • Notification par courrier électronique permet de recevoir un e-mail pour chaque nouveau message dans la salle. Les utilisateurs peuvent choisir de ne pas recevoir d'alertes par courrier électronique.
    • Nombre de messages non lus.

    4
    Ajoutez des jeux de données suggérés à la salle
    L'utilisateur qui a créé une salle publique peut également ajouter un ensemble de jeux de données recommandés, qui sont facilement accessibles par les autres utilisateurs. Salle publique (via le tableau de bord).
      Ce bouton permet de recommander des jeux de données..
