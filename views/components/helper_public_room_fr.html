Public room helper
Une salle publique est un endroit où les utilisateurs peuvent discuter de sujets créés dans l' Agora.
1
À propos de la salle
Au centre, les salles affichent un espace de discussion (activé pour l'utilisation de l'open data). En allant vers l'extrême droite, il existe un espace où c'est possible d'activer des notifications, de visualiser des graphiques et de faire des recherches.
En se deplacent vers l'extrême gauche, une barre d'outils verticale permet de visualiser les fonctionnalités suivantes:
  • Notifications des messages hebdomadaires
    • Un utilisateur peut vérifier le nombre de messages non lus et sur le côté supérieur droit, seuls les messages non lus seront affichés pour tous les jours de la semaine.
  • Recherche
    • Il permet de trouver tous les massages publiés par une personne en particulier.
  • Sentiments
    • Il montre une représentation pour afficher deux valeurs qui représentent le montant, en pourcentage, de commentaires négatifs et positifs.
  • Graphique des utilisateurs
  • Graphique des datalets
    • Le graphique représente deux groupes, c'est-à-dire les datalets sur le côté gauche et les jeux de données sur le côté droit.
  • Notification par e-mail
    • Il permet de suivre l'activité dans chaque discussion avec des notifications par courrier électronique.
    2
    Discussions par sujet

    Une discussion permet de répondre à un commentaire en cliquant sur le bouton "répondre". Chaque fois que vous ajoutez une nouvelle publication dans la discussion, vous pouvez exprimer votre opinion en sélectionnant le bouton Avis :

    • Neutre (Le gris est la couleur de l'avatar dans la salle de discussion).
    • D'accord (Le vert est la couleur de l'avatar dans la salle de discussion).
    • Désaccord (Le rouge est la couleur de l'avatar dans la salle de discussion).
    3
    Graphiques interactifs
    Tout graphique est interactif: vous pouvez naviguer dans les points comme vous le souhaitez, en cliquant sur les points. Tout graphique est informatif: en mettant votre curseur sur un point, les informations relatives sont affichées en haut à droite. Tout graphique est synchronisé: en cliquant sur un point dans le graphique, vous pouvez voir à gauche le contenu correspondant. Il existe deux graphiques:
    • Graphique des utilisateurs : graphique des utilisateurs, où chaque point représente un utilisateur qui participe à la discussion, et une ligne joint aux utilisateurs qu'ils ont des relations d'amitié sur la plateforme SPOD. Si vous mettez le curseur sur un point (pas de clic), vous pouvez voir la relation entre amis parmi les utilisateurs.
    • Graphique des Datalets : Sur le côté droit, le graphique représente deux groupes, sur les datalets sur le côté gauche et les jeux de données sur le côté droit, les jeux de données, les lignes représentent les relations entre eux.
      Si vous apportez le curseurs sur un point (pas de clic), vous pouvez lire le commentaire correspondant et si vous cliquez sur le commentaire à gauche, vous vous déplacez et mettez en surbrillance le datalet correspondant avec un commentaire.
    5
    Partage et prévisualisation d'un datalet
    Vous pouvez utiliser le bouton Datalet pour créer et publier une nouvelle visualisation des jeux de données à tous les niveaux d'une discussion. En cliquant sur le bouton , vous pouvez voir l'aperçu du datalet. Au cours de cette étape, vous pouvez facilement sélectionner l'un des jeux de données suggérés (par le créateur).
