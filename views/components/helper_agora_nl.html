Agora helper
Agora is where each moderator can create a “Public room” to start a new discussion/debate that is open to all the users. Once a room is created, only the administrator can delete the room. On the left side, Agora displays a different views on each tab to organize the Latest, Relevant, Popular and My discussion rooms, finally a “search” feature is available. On the right side, more details are given for each selected discussion room (e.g. description, notifications, statistics and so on) (more details below).
1
Room creation
To create a new room, just press the big "+" (Add) button on the top middle, and insert the subject and a brief description of the room.
2
About rooms (1)
On the left side, Agora displays a different views on each tab to organize the discussion rooms
  • Latest Agora
  • Relevant Agora
  • Popular Agora
  • My Agora shows own created discussion room
  • Search for the title or description of a discussion room
    • 3
    About rooms (2)

    On the right side, more details are given for each selected discussion room:

    • Description shows a short description of the discussion room
    • Statistics section shows Views, Comments, and Datalets. “Views” shows Popularity: how popular is this public room in the SPOD. It depends on how many views has received in total, with respect to the others. Comments and Datalets describe respectively the number of comments and datalets in a discussion room.
    • Users indicates the number of users who have participated in the debate.
    • Suggested Datasets shows all recommended datasets by the user.
    • Email Notification allows to receive e-mail for each new message in the room. Users can opt not to receive the following email alerts.
    • Number of unread messages.

    4
    Add suggested datasets to room
    The user that created a Public Room can also add a set of recommended datasets, that are easily accessible by the participants in that Public Room (via the Controllet).
